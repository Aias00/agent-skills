# Chrome Web Store Image Specs

Use these sizes for Chrome Web Store listing assets:

- Store icon: `128x128` (PNG)
- Screenshots: `1-5` images, each `1280x800` (preferred) or `640x400`
- Small promo tile: `440x280`
- Marquee promo tile (optional): `1400x560`

## Required file naming (recommended)

Put assets under `release/store-assets/`:

- `release/store-assets/icon-128x128.png`
- `release/store-assets/small-promo-440x280.png`
- `release/store-assets/marquee-1400x560.png` (optional)
- `release/store-assets/screenshots/screenshot-1-1280x800.png`
- `release/store-assets/screenshots/screenshot-2-1280x800.png`
- ... up to 5 screenshots
