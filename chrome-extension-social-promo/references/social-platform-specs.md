# Social Platform Specs (Practical Defaults)

Use these as drafting constraints, not hard policy limits.

## X (Twitter)
- Goal: quick hook + one core value + CTA.
- Length target: <= 280 chars.
- Style: concise, high signal, 1-2 hashtags.

## LinkedIn
- Goal: professional launch update with context and value.
- Length target: 600-1200 chars.
- Style: clear paragraphs, short bullet points, practical CTA.

## Reddit
- Goal: transparent "what it does" + why useful.
- Format: title + body.
- Style: avoid hype. Include limitations and use case.

## Telegram
- Goal: concise release broadcast.
- Length target: 300-800 chars.
- Style: direct, readable, include action link.

## Weibo
- Goal: lightweight awareness + quick utility explanation.
- Length target: 120-300 Chinese chars.
- Style: energetic, no exaggerated claims.

## Xiaohongshu
- Goal: scenario-driven recommendation.
- Length target: 250-600 Chinese chars.
- Style: "pain point -> solution -> step -> result".

## Generic Copy Safety
- Avoid unverifiable superlatives or fake metrics.
- Keep claims aligned with actual extension features.
- Mention local processing/privacy only if true in product behavior.
